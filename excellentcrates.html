🎁 ExcellentCrates

Crea cajas de recompensas con llaves para dar premios aleatorios, ítems épicos o dinero a los jugadores.

Lista de Comandos

ComandoDescripción Breve
/cratesAbre el menú principal con la lista de cajas disponibles.
/crates preview [caja]Te permite ver de forma gráfica los premios que contiene una caja.
/crates key give [jugador] [caja] [cant](Admin) Otorga llaves de una caja específica a un jugador.
/crates key giveall [caja] [cant](Admin) Entrega llaves de una caja a todos los usuarios conectados.
/crates claimAbre el inventario de recompensas pendientes por reclamar.
/crates setup(Admin) Abre el menú de configuración interactivo para crear o editar cajas.
/crates reload(Admin) Recarga el plugin y actualiza los porcentajes de premios.
